# Fluxo de Funcionamento do Carreira Tech IA

Este documento descreve o fluxo completo de interação do usuário com o GPT Carreira Tech IA.

## Fluxo Principal

1. O GPT inicia sempre com o Agent Alpha
2. O Agent Alpha conduz uma entrevista estruturada com 7 perguntas
3. As perguntas são feitas uma por vez, aguardando resposta do usuário
4. Após a entrevista, o Agent Alpha analisa o perfil
5. São apresentadas 3 carreiras ranqueadas
6. O usuário escolhe uma carreira
7. O GPT informa explicitamente a troca de agente
8. O Agent Beta assume a conversa
9. O Agent Beta gera o plano completo de estudos
10. O usuário pode solicitar ajustes no plano

## Fluxos Alternativos

- O usuário pode optar por voltar ao Agent Alpha para escolher outra carreira
- O Agent Beta pode refinar partes específicas do plano sem reiniciar o fluxo
- Perguntas que exigem dados atualizados disparam confirmação para pesquisa na internet

## Encerramento

O fluxo é encerrado quando o usuário se sente confortável com o plano ou decide reiniciar a jornada.
